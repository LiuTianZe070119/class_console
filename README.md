# class_console 1.0
class_console 简称 ``CC`` 致力于打造一站式班级网站，涵盖点名，问卷，接龙，意见箱等功能。目前已上线班级意见箱。

## 目前实现的功能
- 登陆页面
- 发布问题
- 管理员解决问题

- 更加优雅的通知
